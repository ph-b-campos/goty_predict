# goty_predict
The objective of this project is to predict whether a video game will receive a Game of the Year (GOTY) nomination. By analyzing a relational database  containing information of market trends, platform metrics, genre popularity, and publisher history, this project is focused on Deep Learning and Feature Engineering.
Data source: [Video Game Sales & Metacritic Intelligence 1980–26] by [Meruva Kodanda Suraj].
Link : https://www.kaggle.com/datasets/meruvakodandasuraj/video-game-sales-and-metacritic-intelligence-198026/data
